# Candy distributor DIY
DIY project which consists of the construction and the implementation of a Candy dispenser using cardboard. The final machine drops a candy when a hand is detected. Please have a look at the video project on my [youtube channel](https://www.youtube.com/watch?v=NE8GYE-GdQc).

## Software Requirements
1. Arduino IDE
3. Fritzing for schematic (optional)

## Hardware Requirements
1. Servo motor
2. IR proximity sensor
3. Green led 5mm
2. Arduino Nano board
3. Cables and connectors
4. USB 2.0 cable to connect Arduino to PC
5. Power supply 9V battery
7. Cardboard and glue pen
